# Support

## Usage Questions

Use GitHub Discussions (if enabled) for setup, usage, and integration questions.

If Discussions are not enabled, open a GitHub issue with the `question` label.

## Bug Reports

Open an issue using the bug report template and include:

- exact command run
- expected behavior
- actual behavior
- OS and Rust version

## Security Issues

Do not open public issues for vulnerabilities.
See [SECURITY.md](SECURITY.md) for private reporting instructions.
